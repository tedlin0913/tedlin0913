### What I work on

- **Autonomous UAV navigation** — onboard stereo visual-inertial SLAM on Jetson Orin Nano, built on PX4 and ROS2
- **3D reconstruction** — Gaussian Splatting pipelines and flight-path strategies for dense scene capture
- **Zero-shot 3D segmentation** — lifting 2D foundation models into 3D instance masks

### Selected work

- **PlantLiftSeg** — zero-shot 3D leaf instance segmentation in greenhouse environments, **mIoU 0.94**. Published at ACPA 2025; under submission to CIGR 2026.
- **RealFlight** — onboard stereo-VIO autonomous flight for in-greenhouse plant scanning, no offboard compute
- **3DGS phenotyping pipeline** — comparing capture strategies for reconstruction fidelity on structurally complex plants
